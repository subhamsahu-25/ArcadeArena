## 🌟 Overview

**Arcade Arena** is a lightweight, no-dependency game arcade that runs entirely in the browser. Pick a game from the home menu and jump straight in — no installs, no frameworks, no libraries. Just pure HTML, CSS and JS.

## 🖱️ How to Play

### ✕ ○ Tic Tac Toe
| Step | Action |
|------|--------|
| 1 | Player X clicks any cell to go first |
| 2 | Player O follows — alternate turns |
| 3 | First to get 3 in a row wins 🏆 |
| 4 | Hit **Play Again** to rematch or **Reset All** to wipe scores |

### ⚡ Tug of War
| Step | Action |
|------|--------|
| 1 | Character 🎯 starts in the middle of the track |
| 2 | Select difficulty — Easy / Medium / Hard |
| 3 | Tap **TAP!** to push it towards your goal 🏆 |
| 4 | If you don't tap in time, ⚡ system pulls it back |
| 5 | Reach either end to decide the winner |
